Internet Engineering Task Force

The Internet Engineering Task Force is an international group of network designers, operators, vendors, and researchers overseeing the advancement of internet architecture and the operation of the internet. IETF Working Groups develop RFC documents that detail computer networking protocols, procedures, programs, and concepts.

Source: IETF Website

Internet Corporation for Assigned Names and Numbers

The Internet Corporation for Assigned Names and Numbers is a non-profit organization that has multiple responsibilities. ICANN determines IP address space allocation, assigns protocol identifiers, manages top-level domain name systems, and is responsible for root server system functions.

Source: ICANN Website

World Wide Web Consortium

The World Wide Web Consortium is a community made up of member organizations, full-time staff, and members of the public work together to develop web standards. These standards define an open platform for application development that enable developers to make experiences available on any device.

Source: W3C Website
